Setup Instructions

  1. Set master password - Choose a strong password to encrypt your credentials
  2. Enter Okta username - Your Okta email address
  3. Enter Okta password - Will be encrypted with your master password
  4. Enable auto-login using the toggle
  5. Navigate to Okta - Auto-login will work automatically!
🔒 Security: Your Okta password is encrypted with your master password. The master password is stored in memory only and cleared when you close Chrome. You'll need to unlock once per browser session.

See MASTER-PASSWORD.md for detailed security information.
